# T02: 在途采购看板与配送记录数据面落地

**优先级**: P0
**状态**: READY
**依赖**: T01

## 目标

补齐 `PROC-INTRANSIT-BOARD` 的在途采购 / 配送节点数据面，并与配送记录页面对齐。

## 技术设计

- 对齐旧系统“配送记录-在途采购”页面
- 锁定真实来源表与状态迁移：
  - 采购申请
  - 配送节点
  - 预计到货时间
  - 当前执行状态
- 优先落 authority/L0，必要时补 L1 聚合视图

## 影响范围

- `authority.procurement.intransit_board`
- 固定报表模板与执行器
- 配送记录页面回归样例

## 验证

- [ ] `PROC-INTRANSIT-BOARD` 远程执行返回 `200`
- [ ] 返回字段包含在途状态、配送节点、预计到货时间

## 完成标准

- [ ] 在途采购看板不再是占位模板
- [ ] 结果可以对齐旧系统配送记录页主要字段
