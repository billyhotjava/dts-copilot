# T03: 到货及时率事实表与指标口径落地

**优先级**: P0
**状态**: READY
**依赖**: T02

## 目标

为 `PROC-ARRIVAL-ONTIME-RATE` 建立真实到货事件事实表/视图，完成时效率指标口径落地。

## 技术设计

- 审计当前 `fact.procurement.order_event` 的声明与空缺
- 锁定“订单创建 / 预计到货 / 实际到货 / 及时率”口径
- 选定：
  - authority 直算
  - 或轻量 `fact` 增量同步
- 补齐固定报表趋势图所需的日期维度聚合

## 影响范围

- ELT / fact 口径
- `PROC-ARRIVAL-ONTIME-RATE`
- 对应 IT 趋势回归样例

## 验证

- [ ] `PROC-ARRIVAL-ONTIME-RATE` 远程执行返回 `200`
- [ ] 返回结果带日期维度与到货及时率指标

## 完成标准

- [ ] 到货及时率从 `DECLARED_ONLY` 变为真实可执行
- [ ] 指标口径在文档和代码中一致
