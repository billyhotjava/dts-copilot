# BG-03: 项目履约语义包

**状态**: READY
**依赖**: BG-02

## 目标

为项目履约主题域建立第一版业务语义包，覆盖项目、客户、合同、点位、项目绿植、履约任务等核心对象。

## 技术设计

- 定义对象：
  - 项目
  - 客户
  - 合同
  - 点位/楼层
  - 项目绿植
  - 履约任务/养护记录
- 明确：
  - 主时间字段
  - 主状态字段
  - 常用筛选字段
  - 核心 join 路径
- 为项目履约高频问句沉淀 few-shot 与 eval case

## 输出

- 项目履约对象字典
- 项目履约同义词集
- 项目履约问句样本
- 项目履约候选指标清单

## 完成标准

- [ ] 项目履约域核心对象全部入库
- [ ] 常见问句可召回正确对象和表集合
- [ ] 至少覆盖项目数、点位数、在场绿植数、履约任务完成率等指标候选
- [ ] 为 BG-05、BG-06、BG-09 提供稳定输入
