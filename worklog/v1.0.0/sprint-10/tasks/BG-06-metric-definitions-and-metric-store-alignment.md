# BG-06: 指标口径与 Metric Store 对齐

**状态**: READY
**依赖**: BG-03, BG-04

## 目标

把两个优先主题域的核心指标正式化，形成 Copilot 和 BI 共用的指标定义。

## 技术设计

- 项目履约域优先指标：
  - 在服项目数
  - 活跃项目数
  - 点位数
  - 在场绿植数
  - 履约任务完成率
- 现场业务域优先指标：
  - 加花次数
  - 换花数量
  - 回收数量
  - 初摆完成数
  - 异常任务数
- 使用 `analytics_metric.metric_json` 作为 v1 指标定义载体
- 与 `MetricLens` 能力保持版本管理和差异比较兼容

## 完成标准

- [ ] 核心指标有统一名称、定义、维度、粒度和 SQL/逻辑表达
- [ ] 同一指标在 Copilot 和 BI 页面使用同一口径
- [ ] 指标定义支持版本迭代
- [ ] 为 BG-07、BG-09 提供直接输入
