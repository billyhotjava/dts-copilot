# BG-09: 轻量 ELT 主题层与增量同步

**状态**: READY
**依赖**: BG-03, BG-04, BG-06

## 目标

在不引入完整数仓平台的前提下，为两个优先主题域建立可被 Copilot 和 BI 共用的轻量 ELT 主题层。

## 技术设计

- v1 主题层对象：
  - `mart_project_fulfillment_daily`
  - `fact_field_operation_event`
- 同步方式：
  - 小步增量
  - watermark 驱动
  - 小时级或更短频率
- 部署方式：
  - 先内嵌在 `dts-copilot-analytics`
  - 暂不引入 dbt / Airflow 作为强依赖

## 设计原则

- 不做全库搬运
- 不做全域指标统一
- 只覆盖项目履约和现场业务两个主题域

## 完成标准

- [ ] 两个主题层对象存在且可增量同步
- [ ] 指标和趋势问题可优先落到主题层
- [ ] 主题层字段和指标定义可追溯到语义资产
- [ ] 为 BG-10 提供可验证的统计型问答路径
