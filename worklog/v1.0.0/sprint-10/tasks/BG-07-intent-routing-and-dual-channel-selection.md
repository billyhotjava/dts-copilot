# BG-07: 意图路由与双通道判定

**状态**: READY
**依赖**: BG-05, BG-06

## 目标

在 Copilot 运行时增加问题路由器，先判断主题域和查询类型，再决定走直连通道还是主题层通道。

## 技术设计

- 先判断主题域：
  - 项目履约
  - 现场业务
  - 其他
- 再判断查询类型：
  - 明细追问
  - 指标统计
  - 趋势分析
  - 诊断解释
- 通道选择：
  - 明细追问 -> 直连通道
  - 指标/趋势 -> ELT 主题层
- 低置信度时先澄清，不直接生成 SQL

## 完成标准

- [ ] 运行时具备主题域识别
- [ ] 运行时具备通道判定
- [ ] 低置信度问题会进入澄清分支
- [ ] 为 BG-08、BG-09、BG-10 提供清晰运行时入口
