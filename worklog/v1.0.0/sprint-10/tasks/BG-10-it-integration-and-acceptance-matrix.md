# BG-10: IT 集成测试与验收矩阵

**优先级**: P2
**状态**: READY
**依赖**: BG-01~09

## 目标

建立本 Sprint 的集成测试目录和验收矩阵，覆盖"预制模板 → 规则路由 → NL2SQL → 权限桥接"的完整链路。

## 技术设计

### 测试维度

1. **预制模板命中** — 高频问句走模板路径，准确率 100%
2. **规则路由准确性** — 业务域判定和视图选择正确
3. **NL2SQL 在视图层的准确率** — 模型基于视图 DDL 生成正确 SQL
4. **枚举词典翻译** — 状态码翻译正确
5. **权限拦截** — 非视图表查询被拒绝
6. **结算域隔离** — 结算问题不走绿植重算
