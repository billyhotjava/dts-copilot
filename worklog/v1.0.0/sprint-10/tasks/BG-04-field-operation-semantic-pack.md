# BG-04: 现场业务语义包

**状态**: READY
**依赖**: BG-02

## 目标

为现场业务主题域建立第一版语义包，覆盖加花、减花、换花、转移、回收、初摆/实摆、执行任务等核心事实对象。

## 技术设计

- 统一现场业务事件词汇：
  - 加花
  - 减花
  - 换花
  - 转移
  - 回收
  - 初摆/实摆
  - 执行任务
- 沉淀事件模型：
  - 事件类型
  - 项目归属
  - 发生时间
  - 状态
  - 数量/件数/点位维度
- 为现场业务高频问句沉淀 few-shot 与 eval case

## 输出

- 现场业务对象字典
- 现场业务事件类型映射
- 现场业务同义词集
- 现场业务问句样本

## 完成标准

- [ ] 现场业务问句不再主要依赖裸表猜测
- [ ] 现场业务事件具备统一命名和分类
- [ ] 至少覆盖加花次数、换花数量、回收数量、初摆完成数等指标候选
- [ ] 为 BG-05、BG-06、BG-09 提供稳定输入
