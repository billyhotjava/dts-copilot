# ER-06: 主题层健康检查与自动降级

**优先级**: P1
**状态**: READY
**依赖**: ER-02, ER-05

## 问题

Sprint-11 文档承诺“主题层不可用时自动回退到视图层”，但当前实现只有静态关键词路由，没有真实健康检查或自动降级。

## 范围

- 定义主题层健康判定
- 将健康状态接入路由
- 将降级原因透传到日志和调试信息

## 建议判定维度

- 同步状态是否为 `COMPLETED`
- 最近同步时间是否在 SLA 内
- 目标表是否存在且有数据
- 关键模板执行是否通过 smoke

## 验收标准

- [ ] 主题层不健康时，自动改走视图层
- [ ] 降级原因可被观测
- [ ] 不会因为主题层异常阻断正常对话
