# T05: 5 类消费者画像与决策路径

**优先级**: P1
**状态**: READY
**依赖**: T01

## 目标

为 T01 采集的真实问句补全"问完之后干什么"——决策路径。这决定 ads 模型要不要带"行动字段"（如联系人电话、责任人、上次催收时间）。

## 5 类消费者

| 角色 | 关心什么 | 决策频率 |
|---|---|---|
| 养护人 | 自己手里的报花单 + 摆位状态 | 实时 / 日 |
| 项目经理 | 自己项目的报花进度 + 客户反馈 | 日 / 周 |
| 业务经理 | 多项目对比 + 异常 + 客户重点 | 周 / 月 |
| 财务 | 待结算 / 坏账 / 销售收入 | 月底 |
| 老板 | KPI / 趋势 / TOP 异常 | 周 / 月 |

## 模板

每条 T01 问句补全：

```
问句：万象城项目本月加摆了多少？

消费者：王经理（万象城项目负责人）
角色：项目经理
频次：每周问 1-2 次
紧迫度：中

下游动作：
  → 加摆 < 5 盆 → 不动作
  → 加摆 5-20 盆 → 关注库存
  → 加摆 > 20 盆 → 通知采购备货 + 通知养护人增援

需要的字段：
  - 加摆数量（核心）
  - 加摆金额（核心）
  - 加摆品种分布
  - 客户对接人电话（行动！）  ← 当前 ads 没有
  - 责任养护人               ← 当前 dws_curing_user_monthly 有
  - 库存预警                 ← 当前没有，sprint-23 库存域

问句变体（同一意图，不同问法）：
  - 万象城本月加了多少植物？
  - 王经理项目这周加摆几次？
  - 这个月万象城新植入金额？
```
