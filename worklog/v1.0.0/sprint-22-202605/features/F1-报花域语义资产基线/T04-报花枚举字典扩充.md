## 4 类报花枚举

### flower_biz.status（报花单状态，7 + 2 = 8 值）

| enum_value | label | description | sort_order | is_active |
|---|---|---|---|---|
| 20 | 草稿 | 未提交 | 10 | true |
| 1 | 审核中 | 等待项目经理 / 业务经理审核 | 20 | true |
| 21 | 驳回 | 退回草稿重编 | 30 | true |
| 2 | 备货中 | 已审，等采购+配送+入库 | 40 | true |
| 3 | 核算中 | 入库完成，等财务核算 | 50 | true |
| 4 | 待结算 | 核算完成，等月底结算 | 60 | true |
| 5 | 已结束 | 全流程完成 | 70 | false |
| -1 | 作废 | 任意状态可作废 | 99 | false |

### flower_biz.biz_type（业务类型，10 个有效值）

| enum_value | label | description | sort_order |
|---|---|---|---|
| 1 | 换花 | 同摆位替换绿植，租金可能变化 | 10 |
| 2 | 加花 | 增加新绿植，触发采购 + 租金 +正 | 20 |
| 3 | 减花 | 移除绿植，触发回收 + 租金 -负 | 30 |
| 4 | 调花 | 跨摆位 / 跨库房调拨 | 40 |
| 6 | 坏账 | 客户欠款核销，金额 -负 | 50 |
| 7 | 售花 | 销售绿植，走 SaleAccount 链 | 60 |
| 8 | 赠花 | 无偿提供，走 SaleAccount 链 | 70 |
| 10 | 配料 | 添加辅料 | 80 |
| 11 | 加盆架 | 增加盆架 | 90 |
| 12 | 减盆架 | 减少盆架 | 100 |

### flower_biz.recovery_type（回收去处，3 值）

| enum_value | label | description |
|---|---|---|
| 1 | 报损 | 绿植无法继续使用 |
| 2 | 回购 | 客户付费保留绿植 |
| 3 | 留用 | 库房留用，下次再摆 |

### flower_biz.change_type（变更类型，4 值）

| enum_value | label | description |
|---|---|---|
| 1 | 销售金额变更 | totalAmount 修改 |
| 2 | 库房物品类型变更 | good_price_id 修改 |
| 3 | 成本变更 | cost 修改 |
| 4 | 起租减租变更 | start/end_lease_time 修改 |
