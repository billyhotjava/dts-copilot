## 真人联调清单（5 类消费者）

### A. 养护人视角

```
1. 打开 ChatPanel
2. 输入：我手里这周的报花单
3. 期望：列表，按养护人=当前用户过滤
4. 输入：李师傅本月加了多少花
5. 期望：单数 + 金额 + 品种分布
```

### B. 项目经理视角

```
1. 输入：万象城最近的报花单
2. 输入：我项目本月撤摆多少
3. 输入：审核中超过 3 天的，提醒一下
```

### C. 业务经理视角

```
1. 输入：本月新签客户的首次报花
2. 输入：销售前 10 项目
3. 输入：哪些客户最近坏账多
```

### D. 财务视角

```
1. 输入：待结算的报花单数
2. 输入：本月坏账金额
3. 输入：销售收入 vs 租赁收入对比
```

### E. 老板视角

```
1. 输入：本月加摆撤摆净增减
2. 输入：与上月相比，加摆下降的项目
3. 输入：养护人工作量 TOP 5
```
