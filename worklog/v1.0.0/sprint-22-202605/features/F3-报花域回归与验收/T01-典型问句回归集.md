## 回归集结构

`it/flowerbiz-query-regression.md`，每条：

```
## R1: 本月加摆撤摆净增减

- 问句：本月各项目加摆撤摆净增减多少？
- 期望路由：domain=flowerbiz, template=FB-Q-LEASE-NET-MONTHLY
- 期望视图：xycyl_ads.xycyl_ads_flowerbiz_lease_summary
- SQL 关键特征：
  - 必有 `WHERE 业务月份 = ...`
  - 必有 `GROUP BY 项目, 客户`
  - 必有 ROUND(..., 2)
  - 不应出现 `SUM(biz_total_rent)`（直接聚合 13 bizType 是反例）
- 期望结果（远程 ai.yuzhicloud.com）：
  - 行数：当月有报花的项目数
  - Top1 净增金额：≥ Top2
  - 列名：项目, 客户, 加摆, 撤摆, 净增, 单数
- 反例（必须不出现）：
  - 用 SUM(biz_total_rent) 不分组
  - 用 create_time 作业务时间
```
