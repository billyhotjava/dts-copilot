### 功能测试用例

| 编号 | 场景 | 操作 | 期望结果 |
|------|------|------|---------|
| V-01 | 正常语音输入 | 点击麦克风 → 说"查一下中石油项目的绿植" | 文本出现在输入框 |
| V-02 | 中间结果 | 说话过程中观察输入框 | interim text 实时更新 |
| V-03 | 手动停止 | 录音中点击麦克风 | 停止录音，final text 填入 |
| V-04 | 自动停止 | 说完后停顿 2-3 秒 | 自动停止，final text 填入 |
| V-05 | 超时停止 | 持续录音 30 秒 | 自动停止 |
| V-06 | 权限被拒 | 点击麦克风 → 拒绝权限 | 显示"麦克风权限被拒绝"提示 |
| V-07 | 编辑后发送 | 语音输入后修改文本 → 发送 | 修改后的文本被发送 |
| V-08 | 连续使用 | 第一次语音 → 发送 → 第二次语音 | 两次都正常工作 |
| V-09 | 与打字混用 | 打字一半 → 语音输入 | 语音文本追加/替换 |
| V-10 | 发送中禁用 | 消息发送中点击麦克风 | 按钮禁用不响应 |
