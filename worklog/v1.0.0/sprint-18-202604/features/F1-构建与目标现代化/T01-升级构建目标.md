# T01: 升级构建目标到 esnext

**优先级**: P0
**状态**: READY
**依赖**: 无

## 目标

将构建目标从 Chrome 95 升级到 esnext / Chrome 120+，减少转译代码量。

## 技术设计

### 1. package.json browserslist

```diff
- "production": ["Chrome >= 95"]
+ "production": ["Chrome >= 120", "Edge >= 120"]
```

### 2. vite.config.ts

```diff
- const legacyBuild = process.env.LEGACY_BROWSER_BUILD === '1';
- const esbuildTarget = legacyBuild ? 'chrome95' : 'chrome109';
+ const esbuildTarget = 'esnext';

  build: {
-   target: esbuildTarget,
+   target: 'esnext',
  }
```

移除 `LEGACY_BROWSER_BUILD` 环境变量相关的所有逻辑。

### 3. tsconfig.json

```diff
- "target": "ES2022",
- "lib": ["ES2022", "DOM", "DOM.Iterable"],
+ "target": "ES2024",
+ "lib": ["ES2024", "DOM", "DOM.Iterable"],
```

## 影响范围

- `package.json` — browserslist
- `vite.config.ts` — build target
- `tsconfig.json` — target 和 lib

## 完成标准

- [ ] `npx vite build` 成功
- [ ] `npx tsc --noEmit` 成功
- [ ] 构建产物中无 Chrome 95 降级代码
