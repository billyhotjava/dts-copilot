# T02: 核心样式文件嵌套改写

**优先级**: P0
**状态**: READY
**依赖**: T01

## 目标

将前 10 大 CSS 文件改写为原生嵌套语法，减少重复选择器。

### 改写规则

```css
/* 改写前 */
.copilot-chat__header { ... }
.copilot-chat__header-title { ... }
.copilot-chat__header-actions { ... }
.copilot-chat__header-actions button { ... }

/* 改写后 */
.copilot-chat__header {
    ...

    &-title { ... }

    &-actions {
        ...

        & button { ... }
    }
}
```

### 不改写的情况

- 全局 reset/normalize 样式
- `tokens.css` 和 `utilities.css`（变量和工具类不适合嵌套）
- 仅 1-2 个规则的小文件
