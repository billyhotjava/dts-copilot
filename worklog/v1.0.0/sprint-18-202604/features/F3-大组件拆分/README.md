## 拆分原则

1. **按职责拆分**：一个文件只做一件事
2. **提取 hooks**：复杂状态逻辑提取为 `use*.ts`
3. **提取子组件**：大 render 函数拆分为独立组件文件
4. **提取常量**：配置、映射表、模板数据移到 `*.constants.ts`
5. **保持 API 兼容**：对外的导出接口不变
