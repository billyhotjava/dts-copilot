# Sprint-11 IT

本目录用于保存 `Sprint-11: 轻量 ELT 主题层与增量同步` 的集成测试计划和验收矩阵。

## 目标

验证主题层的同步正确性、查询性能和路由判定：

- 增量同步：watermark 驱动、幂等、状态码翻译
- 查询性能：主题层 < 1s，视图层 < 3s
- 路由判定：趋势类走主题层，明细类走视图层
- 降级机制：主题层不可用时回退到视图层

## 覆盖范围

- 2 个主题层表：`mart_project_fulfillment_daily`, `fact_field_operation_event`
- 同步引擎：EltSyncScheduler, EltSyncService, watermark 管理
- 路由扩展：DataLayer 判定
- 监控 API：状态查询、手动触发、重置

## 测试维度（详见 EL-08）

| 维度 | 用例数 | 说明 |
|------|--------|------|
| 同步正确性 | 5 | 全量/增量/幂等/翻译/恢复 |
| 查询性能 | 4 | 主题层 vs 视图层对照 |
| 路由正确性 | 4 | 趋势→主题层, 明细→视图层 |
| 降级验证 | 3 | 空表/延迟/配置关闭 |

## 约束

- 性能基准基于万级数据量
- 主题层同步频率默认每小时
- 不验证实时性（这不是流式处理）
