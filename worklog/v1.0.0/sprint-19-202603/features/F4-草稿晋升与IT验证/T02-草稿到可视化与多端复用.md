# T02: 草稿到可视化与多端复用

**优先级**: P1
**状态**: DONE
**依赖**: T01

## 目标

让草稿可顺畅进入图表、仪表盘、大屏和报告工厂。

## 技术设计

- 复用现有多入口 fixed-report / visualization 经验
- 将 draft 作为创建可视化的统一来源
- 仪表盘优先复用/晋升草稿对应查询卡片
- 报告工厂优先复用草稿关联会话作为报告来源
- 大屏优先复用草稿问题语义与展示建议生成 AI prompt

## 影响范围

- visualization flow
- dashboard / screen / report-factory entry

## 验证

- [x] 多入口 smoke

## 完成标准

- [x] 草稿可成为多端分析资产的共同起点
